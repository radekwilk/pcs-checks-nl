How to find SIZE (amount cooked)
  1. You will need Freshness Standard Report
  2. Find column called "Size".
  3. Then go to the bottom of the report and look for total number
  4. Enter this number into web form, under 'Size' field
How to find QPM WASTE (QMP input waste)
  1. You will need Freshness Standard Report
  2. Find column called "Waste".
  3. Then go to the bottom of the report and look for total number
  4. Enter this number into web form, under 'Waste' field
How to find UNACCOUNTED FOR
  1. You will need Freshness Standard Report
  2. Find table called "Unaccounted For Sale".
  3. Then look for Total number for Unaccounted For Sale
  4. Enter this number into web form, under 'Unaccounted for' field
How to find PHYSICAL WASTE
  1. You will need Daily Scorecard Report
  2. Find product name on the left hand side of the report
  3. Look for "Waste" column in the report. In exapmle above, you will look for:
    • Chicken original (pieces)
    • Waste = 23
  4. Enter this number into web form, under 'Physical waste' field
How VARIANCE pieces is calculated
  1. This filed is calculated for you.
  2. This filed is to show you difference between product wasted in QPM system and product wasted physicaly (DailyScorecard).
  3. Based on this number, % of variance will be calculated.
How VARIANCE % is calculated
  1. This filed is calculated for you.
  2. This filed is to show you a variance(%) between physical and theoretical waste. Variance in pieces will be devided by number of pieces sold
  3. If this filed is greater than 10%, it indicates that waste in QPM didn't match physical waste.
  4. If variance is greater than 10% for TWO or more products, it's QPM abuse.
