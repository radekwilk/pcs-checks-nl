How to find AMOUNT COOKED
  1. You will need Gold Standard Report
  2. Look for column called 'Amount Cooked (pc)'
  3. Then go to the bottom of the report and look for total number
  4. Enter this number into web form, under 'Amount cooked' field
How to find Input Trashed (EPCS input waste)
  1. You will need Gold Standard Report
  2. Find column called "Input Trashed (pc)".
  3. Then go to the bottom of the report and look for total number
  4. Enter this number into web form, under 'Input trashed' field
How to find Input Trashed (EPCS input waste)
  1. You will need Gold Standard Report
  2. Unaccounted for is under total numbers line, at the bottom of the report (please see print screen)
  3. Many times unaccounted for is coming from last cooked batch. Store do not waste product after store is closed.
  4. Lets explain example from print screen (blue squers):
    • Last batch of Fillets cooked: 5
    • Quantities Sold: 1
    • Input Trashed: 0
    • Therefore Unaccounted For = 4 (5-1 = 4)
    • If Unaccounted for is coming from Last Batch do not enter it into web form. Leave it empty or enter 0
  5. If it isn't coming from last batch enter it into web form, under 'Unaccounted For' field
How to find PHYSICAL WASTE
  1. You will need Daily Scorecard Report
  2. Find product name on the left hand side of the report
  3. Look for "Waste" column in the report. In exapmle above, you will look for:
    • Chicken original (pieces)
    • Waste = 23
  4. Enter this number into web form, under 'Physical waste' field
How VARIANCE pieces is calculated
  1. This filed is calculated for you.
  2. This filed is to show you difference between product wasted in EPCS system and product wasted physicaly (DailyScorecard).
  3. Based on this number, % of variance will be calculated.
How VARIANCE % is calculated
  1. This filed is calculated for you.
  2. This filed is to show you a variance(%) between physical and theoretical waste. Variance in pieces will be devided by number of pieces sold
  3. If this filed is greater than 10%, it indicates that waste in EPCS didn't match physical waste.
  4. If variance is greater than 10% for TWO or more products, it's EPCS abuse.
